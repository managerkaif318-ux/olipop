# **App Name**: Olipop Flavorscape

## Core Features:

- Cinematic Scroll-Controlled Parallax: Implement a full-screen hero section where WebP image sequences play forward/reverse based on user scroll, providing a smooth, cinematic experience. Ensures smooth playback tied to scroll position and efficient frame loading.
- Dynamic Drink Variant Display: Render drink-specific content (name, subtitle, description, CTAs) and adjust the hero section's 'theme color' dynamically based on the selected variant.
- Intuitive Variant Navigation: Enable users to switch between drink variants via 'PREV'/'NEXT' controls, triggering seamless content updates, theme color changes, and WebP sequence swaps with elegant fade transitions for text.
- Comprehensive Loading Experience: Display a full-screen loading overlay with brand logo and a horizontal progress bar during initial WebP sequence preload, and subtle indicators during variant switching to prevent flicker and enhance user experience.
- Content Management for Drink Variants: Allow administrators to easily define and update drink variant data including names, subtitles, descriptions, customizable theme colors, and paths for WebP image sequences.
- Dynamic Dark/Light Mode Toggle: Provide a functional toggle for switching between predefined dark and light mode aesthetics, adjusting background colors, text colors, and component styles accordingly, while defaulting to a dark cinematic look.
- Integrated Information and Call-to-Actions: Include additional static content sections like a responsive FAQ accordion, a final prominent Call-to-Action with supporting text and buttons, and a clean footer with brand links and social icons.

## Style Guidelines:

- Dark color scheme chosen for a cinematic and premium feel. Primary accent color (Cherry Red: #E01E37) reflecting the vibrant 'Cherry Vanilla' flavor and supporting a 'Playful, Modern, Vibrant' mood. This primary color would dynamically change with each drink variant. Background color: A very dark, subtle reddish charcoal (#1A1515) to maintain the cinematic near-black base. Accent color: A soft, bright pink-magenta (#F28CCD) to provide a secondary vibrant highlight, suitable for dynamic indicators and subtle visual cues, complementing the primary red.
- Headline font: 'Space Grotesk' (sans-serif) for large, bold, uppercase text, lending a modern, techy, and functional feel. Body font: 'Inter' (sans-serif) for paragraphs and general readability, offering a clean, neutral, and objective appearance. Together, they balance expressive headlines with clear body content.
- Utilize minimal and monochrome social media icons (Twitter/X, Instagram, Facebook etc.) in the footer and bottom center of the hero. These should be clean vectors, aligned with the brand's sleek and modern aesthetic.
- A single-page layout featuring a full-screen parallax hero, dynamic left-aligned text overlay, an empty central visual focus area, and a vertically-centered right-side variant navigation. Subsequent sections include a FAQ accordion, a distinct final Call-to-Action block, and a consistent, clean footer. Elements are designed to maximize readability against the animated background.
- Smooth, performance-optimized scroll-controlled WebP sequence playback for a cinematic background. Seamless fade-in/fade-out transitions for text content when switching between drink variants. Loading indicators for initial page load and subtle animations when changing variants to provide clear user feedback without interrupting flow.
